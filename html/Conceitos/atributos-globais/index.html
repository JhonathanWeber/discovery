class

Para que é utilizado o atributo class?

São utilizados para definir uma classe de tag, para sofrer interações através do css ou javascript

id

Para que é utilizado o atributo id?

É utilizado para marcar um tag como unica e deve se utilizar nomes únicos para cada id, ou seja 1 por pagina de navegação.

tabindex

Para que é utilizado o atributo tabindex?

permite ordenação na nevegação quando o a tecla tab no teclado for pressionada, muito utilizada dentro de input no form.

hidden

Para que é utilizado o atributo hidden?

Exclui a tag ao qual foi aplicada está propriedade, para meu view na DOM

title

Para que é utilizado o atributo title?

utilizando para quando houver um hover no tag ele mostrara um pequeno pop-up como um titulo nele.

contenteditable

Para que é utilizado o atributo contenteditable?

Utilizado para quando eu quero deixar meu texto ou conteudo de forma editavel na minha view do body.

style

Para que é utilizado o atributo style?

utlizada para aplicar estilos css diretamente a uma tag, não é mais uma pratica usual.

data-*

Para que é utilizado o atributo data-*?

utilizada para integração com javascript ou até mesmo css, parece com a tag class.
